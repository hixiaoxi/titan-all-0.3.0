Apache Lucene is a high-performance, full-featured text search engine library written entirely in Java. It is a technology suitable for nearly any application that requires full-text search, especially cross-platform. Apache Lucene is an open source project available for free download. — Apache Lucene Homepage

Titan supports Lucene as a single-machine, embedded index backend.






Feature Support

Full-Text

Supports the Text.CONTAINS condition to search for text properties that contain a given word.

Geo

Supports the Geo.WITHIN condition to search for points that fall within a given geographic shape. Only supports points for indexing and circles and boxes for querying.

Numeric Range

Supports all numeric comparisons in Compare.

Option Description Value Default Modifiable
